## Cara Menjalankan Notebook

### Opsi 1 — Google Colab

1. Buka notebook `.ipynb` yang tersedia pada repository GitHub.
2. Klik **Open in Colab** atau upload notebook ke Google Colab.
3. Jalankan setiap cell secara berurutan.

### Opsi 2 — Lokal (Jupyter Notebook)

Clone repository:

```bash
git clone https://github.com/nidaank/DataScience_250401020123_Nida-an-Khafiyya.git
cd DataScience_250401020123_Nida-an-Khafiyya
```

Install dependency:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn jupyter
```

Jalankan Jupyter Notebook:

```bash
jupyter notebook
```

Kemudian buka notebook yang ingin dijalankan.
